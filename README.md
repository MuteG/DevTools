# DevTools 
工作中常用到的一些实用工具。  
  
本工具需要4.0及以上版本.NET Framework支持。 
  
在计算机没有安装IDE的情况下，按照以下步骤操作，即可生成可执行程序：  
1、下载最新代码。  
2、进入代码文件夹的根目录。  
3、执行根目录下的“build.bat”。  
4、在当前目录找到“Release”文件夹。  
5、执行“Release”文件夹内的“DevTools.exe”。  
6、开始使用。  

### 功能简介
* 批量删除文件
* 以解决方案/工程/文件夹为单位，统计代码行数
* 数据库实用工具（暂时无法使用）
